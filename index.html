FJORD

Real-time API Proxy for Kafka

Fjord is an open-source, real-time API proxy for Apache Kafka.

AWS ECS
AWS Elasticache for Redis
Docker
EventSource API

Fjord exposes customizable, real-time API endpoints to allow web clients to securely stream from Kafka.

Through Fjord’s CLI, developers can easily and quickly deploy dozens of components on AWS to offload the streaming responsibilities to Fjord’s scalable infrastructure.

Fjord is open-source, scalable and simple to deploy. You also have full ownership of the data coming in and out of Fjord.
